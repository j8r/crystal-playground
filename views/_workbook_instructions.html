Workbook

This page becomes a workbook like /about by running crystal play in a directory with a workbook.html file.

Simply place crystal code blocks in <pre class="playground"></pre> tags.

For example:

<pre class="playground">
[0x679bff14d, 0x2a4, 0x5cd88d86, 0xbe0ed]
  .map(&.to_s(36)).join(" ")</pre>

becomes

[0x679bff14d, 0x2a4, 0x5cd88d86, 0xbe0ed]
  .map(&.to_s(36)).join(" ")
